# Customer Email Response Rubric

Score each response 1-5 on these dimensions:

| Dimension | 1 (Poor) | 3 (Adequate) | 5 (Excellent) |
|-----------|----------|--------------|---------------|
| **Empathy** | Ignores emotion | Acknowledges | Validates + connects |
| **Specificity** | Generic template feel | Addresses the issue | References exact details |
| **Solution** | No solution offered | Vague next steps | Concrete action with timeline |
| **Tone Match** | Wrong tone for situation | Acceptable | Perfectly calibrated |
| **Completeness** | Misses key concerns | Addresses main issue | Addresses all points raised |
| **Brevity** | Too long or too short | Acceptable length | Exactly right â€” no waste |

**Total possible: 30 points per email, 150 points across all 5.**
