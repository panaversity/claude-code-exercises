## Format
- Executive summary (2-3 sentences)
- Key metrics table with month-over-month comparison
- Highlights section (top 3 wins)
- Challenges section (top 3 issues)
- Recommendations (2-3 actionable items)
- Outlook paragraph

## Rules
- Lead with the single most important number
- Use percentage changes, not just absolute numbers
- Keep total report under 500 words
- Professional tone, no jargon
- Recommendations must be specific and actionable
