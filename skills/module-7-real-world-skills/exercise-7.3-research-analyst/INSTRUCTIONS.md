# Exercise 7.3 — Production Skill: Research Analyst

## Goal
Build a skill that conducts structured research on a topic and produces a comprehensive analysis document.

## The Skill Must
1. Accept a research question or topic
2. Define the research scope (what to cover, what to exclude)
3. Gather information (from provided documents or web search)
4. Synthesize findings into a structured analysis
5. Identify key themes, agreements, and disagreements
6. Produce a research memo with citations
7. Include a "confidence assessment" — how reliable are the findings?

## Provided Materials
- `research-questions/` — 3 research questions to test with
- `source-documents/` — Documents to use as source material
- `analysis-template.md` — Expected output structure

## Build Process
1. Create `research-skill/SKILL.md`
2. Include a reference to the analysis template
3. Test on research question 1 using the provided source documents
4. Test on research question 2 (Claude should search the web)
5. Test on research question 3 (requires combining sources + web)

## Quality Bar
A senior analyst should be able to use the output as a starting point for a real report.
